# DSA Hint Engine - Starter Kit

Build your first real local LLM application. A FastAPI helper that uses Ollama to deliver progressive, non-spoiling DSA feedback.

## Run in 5 Commands

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
uvicorn app.main:app --reload
```
